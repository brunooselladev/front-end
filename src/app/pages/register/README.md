# Registro

Estructura de pantallas y flujos para el proceso de registro de usuarios.

- **Necesito ayuda**
  - Para mí: registro en 2 pasos (datos básicos, datos adicionales)
  - Para otro: registro en 2 pasos (datos referente, datos persona a ayudar)
- **Ofrezco ayuda**
  - Referente afectivo
  - Efector de salud
  - Agente comunitario

Todos los usuarios deben ser aprobados por el administrador.
